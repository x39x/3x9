Google 高级搜索语法，帮助我们更精准的搜索


###  常用搜索技巧

| 操作符      | 用法示例           | 说明                                     |
|-------------|--------------------|------------------------------------------|
| `site:`     | `site:example.com` | 只搜索某个网站中的内容                   |
| `inurl:`    | `inurl:login`      | 搜索网址中含有特定关键词的页面           |
| `intitle:`  | `intitle:教程`     | 搜索标题中含有关键词的页面               |
| `filetype:` | `filetype:pdf`     | 限定搜索特定文件类型（如 PDF, DOC, PPT） |
| `""`        | `"精确短语"`       | 精确匹配短语                             |
| `-`         | `-知乎`            | 排除某个关键词或网站                     |
| `OR`        | `前端 OR 后端`     | 搜索包含两个关键词之一的页面             |
| `*`         | `"Java * 教程"`    | 星号为通配符，可匹配任意词               |


### 🛠 进阶搜索组合示例

- `site:gov.cn filetype:pdf 疫情防控`  
  👉 只找政府官网上发布的 PDF 疫情资料。

- `intitle:招聘 inurl:2025`  
  👉 找标题包含“招聘”且网址中含有“2025”的页面。

- `"Linux 教程" -CSDN -简书`  
  👉 精确查找 Linux 教程但排除 CSDN 和 简书 的结果。


### 💡 其他冷门但有用的技巧

| 技巧                 | 用法                        | 说明                                                         |
|----------------------|-----------------------------|--------------------------------------------------------------|
| `cache:`             | `cache:example.com`         | 查看该网页的缓存版本                                         |
| `related:`           | `related:example.com`       | 查找与某网站类似的网站                                       |
| `define:`            | `define:hypervisor`         | 查询某词的定义（相当于查词典）                               |
| `AROUND(n)`          | `"人工智能" AROUND(5) 医疗` | 限定两个关键词在文本中距离不超过 n 个词                      |
| `before:` / `after:` | `AI before:2020`            | 查找某时间之前（或之后）的结果（配合 Google 英文界面更有效） |


想找某类隐藏页面、后台、接口等，也可以组合：
```bash
site:example.com inurl:admin
site:example.com filetype:json
```
